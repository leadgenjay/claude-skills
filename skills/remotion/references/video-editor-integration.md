# Video-Editor Project Integration

Cross-project reference for the standalone video editing pipeline.

## Project Location

```
~/Documents/Tech & Dev/Studio Apps/video-editor/
```

## Architecture

**"Python detects, FFmpeg extracts, Remotion composes"**

| Layer | Technology | Purpose |
|-------|-----------|---------|
| Video Composition | Remotion 4.0 (React) | All visual rendering, text overlays, captions, blur regions |
| Video Processing | FFmpeg 8.0.1 | Normalization, audio extraction, silence detection, concatenation |
| Audio Denoise | DeepFilterNet (Python) | ML-based background noise removal |
| Transcription | Whisper CPP | Word-level timestamps (medium.en model) |
| OCR | Tesseract | Frame text extraction for PII detection |
| TTS | ElevenLabs | Voice cloning for ad voiceovers |

## Three Remotion Compositions

| Composition | Format | Use Case |
|-------------|--------|----------|
| **ScreenshareEdit** | 1920x1080 / 1080x1920 / 1080x1080 | Standard screenshare editing — segments, captions, blur, overlays |
| **ShortFormEdit** | 1080x1920 only | Talking-head videos with storyboard, AI enrichments, B-roll |
| **TextAd** | 1080x1920 | Kinetic typography ads with voice sync + music |

### Pipeline Options

| Flag | Purpose |
|------|---------|
| `--shortform` | Vertical + pop captions + music + tight cuts |
| `--talking-head` | AI-driven short-form with visual enrichments |
| `--redaction` | OCR + PII scanning |
| `--no-denoise` | Skip audio denoising |
| `--no-captions` | Skip captions |
| `--music` | Add background music |
| `--preset landscape\|vertical\|square` | Output format |
| `--whisper-model tiny\|base\|small\|medium.en\|large` | Transcription accuracy |

## Pipeline Stages (Standard Video)

probe → normalize → extract_audio → extract_frames → transcribe → detect_filler → detect_silence (Silero VAD) → detect_repeats → analyze_transcript → rough_cut → denoise → render → finalize

**Talking-Head Pipeline:**
probe → extract_audio → transcribe → storyboard → discovery (AI) → plan_review → denoise → build_manifest → render

## Critical Gotchas

1. **FFmpeg AAC + sample_fmt** — Never use `-sample_fmt s16` with AAC. Use `-c:a aac -b:a 128k`
2. **Whisper.cpp paths** — Always pass absolute paths (`path.resolve()`). Relative paths silently fail
3. **Whisper.cpp sample rate** — Requires 16kHz WAV input. Create separate `audio-16k.wav` at `-ar 16000`
4. **Whisper model** — Only `medium.en` installed. Default to `medium.en`, not `base`
5. **Python venv** — Always use `.venv/bin/python3`. System python lacks uvicorn/dependencies
6. **FFmpeg concat filter order** — Inputs must be interleaved `[v0][a0][v1][a1]...`, NOT grouped
7. **Remotion video serving** — Hardlink/copy video into bundle directory after `bundle()`. Override `sourceFile` to just filename
8. **Remotion dynamic duration** — Use `calculateMetadata` in Root.tsx. Never hardcode `durationInFrames`
9. **Remotion imports** — Files in component tree must use extensionless imports (webpack doesn't resolve `.js` → `.tsx`)
10. **FFmpeg drawtext missing** — No libfreetype/libass. All text MUST use Remotion, not FFmpeg text rendering
