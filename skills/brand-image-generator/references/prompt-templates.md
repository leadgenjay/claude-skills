## Customization Tips

### Adding Specific Elements
Insert after the main subject:
```
[base prompt], with [specific element], [additional element]
```

### Adjusting Color Emphasis
Add at end:
```
[base prompt], [color] (#hex) as dominant accent color
```

### Changing Mood
Prepend mood descriptor:
```
[mood] [base prompt], [lighting descriptor]
```

**Mood options:** energetic, calm, professional, innovative, trustworthy, modern

### Increasing/Decreasing Detail
- More detail: Add "intricate details", "complex", "detailed"
- Less detail: Add "simple", "minimal", "clean", "uncluttered"
