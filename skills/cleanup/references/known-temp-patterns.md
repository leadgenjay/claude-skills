# Known Temp File Patterns

Project-specific temporary files that accumulate during work sessions. Use these patterns for detection in the `temp` cleanup category.

## GHL (GoHighLevel) Artifacts

Generated by `scripts/ghl-*.mjs` during workflow extraction and inspection sessions.

| Pattern | Origin Script | Purpose | Safe to Delete |
|---------|--------------|---------|----------------|
| `ghl-*.png` (root) | `ghl-screenshot-scroll.mjs`, `ghl-extract.mjs` | Workflow screenshots | Yes |
| `ghl-*.json` (root) | `ghl-extract.mjs`, `ghl-inspect*.mjs` | Extracted workflow data | Yes |
| `ghl-screenshots-wf0/` | `ghl-screenshot-scroll.mjs` | Paginated scroll captures | Yes |

## Playwright Browser Caches

| Pattern | Origin | Purpose | Safe to Delete |
|---------|--------|---------|----------------|
| `.auth/ghl-playwright/` | Playwright `launchPersistentContext()` | Browser session/cache for GHL automation | Yes (81MB Chromium cache) |
| `.auth/*/` (other) | Various Playwright scripts | Persistent browser contexts | Check individually |

## Script Output Artifacts

| Pattern | Origin | Purpose | Safe to Delete |
|---------|--------|---------|----------------|
| `output/ads/**` | `generate-cold-email-ads.mjs` | Generated ad images | Yes (gitignored, regeneratable) |
| Root `*.png` / `*.jpg` | Various scripts | Debug screenshots | Verify not a needed asset first |

## Detection Command

```bash
ls -la ghl-*.{png,json} 2>/dev/null
ls -la ghl-screenshots-*/ 2>/dev/null
du -sh .auth/*/ 2>/dev/null
ls -la *.png *.jpg 2>/dev/null | grep -v node_modules
```
