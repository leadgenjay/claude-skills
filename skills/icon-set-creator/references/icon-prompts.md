# Icon Prompt Templates

Ready-to-use prompts organized by style and category for consistent icon sets.

---

## Style Templates

### Flat/Minimal Style (Recommended Default)

**Base Prompt:**
```
Minimalist flat icon of [SUBJECT], blue (#2563eb) monochrome, clean white background, simple geometric shapes, no gradients, no shadows, centered composition, consistent visual weight, professional vector style, high quality
```

**Variations:**
- Light blue: `light blue (#93c5fd) monochrome`
- Navy: `navy blue (#1e3a8a) monochrome`
- Gray: `slate gray (#64748b) monochrome`

---

### Modern Gradient Style

**Base Prompt:**
```
Modern gradient icon of [SUBJECT], smooth gradient from navy (#1e3a8a) to bright blue (#3b82f6), subtle 3D depth, soft drop shadow, light gray (#f8fafc) background, contemporary tech style, high quality
```

**Variations:**
- Blue to cyan: `gradient from blue (#2563eb) to cyan (#06b6d4)`
- Purple to blue: `gradient from purple (#7c3aed) to blue (#3b82f6)`
- Dark to light: `gradient from dark blue (#0f172a) to light blue (#93c5fd)`

---

### Outlined/Line Style

**Base Prompt:**
```
Clean outlined icon of [SUBJECT], thin blue (#2563eb) strokes, no fill, consistent 2px line weight, rounded corners, white background, minimalist line art, centered, high quality
```

**Variations:**
- Thicker lines: `consistent 3px line weight`
- Sharp corners: `sharp corners, no rounding`
- Dotted style: `dashed line style`

---

### Duotone Style

**Base Prompt:**
```
Duotone icon of [SUBJECT], primary blue (#2563eb) with light blue (#93c5fd) secondary tone, layered design creating depth, clean white background, modern two-tone aesthetic, high quality
```

**Variations:**
- High contrast: `primary navy (#1e3a8a) with light blue (#bfdbfe)`
- Subtle: `primary blue (#3b82f6) with very light blue (#eff6ff)`

---

### Filled/Solid Style

**Base Prompt:**
```
Solid filled icon of [SUBJECT], bold blue (#2563eb) fill, clean white background, simple geometric shape, no gradients, strong silhouette, centered, professional quality, high resolution
```

## Batch Generation Checklist

When generating a full icon set:

1. **Copy base prompt** - Use exact same base for all icons
2. **Only change subject** - Keep everything else identical
3. **Generate at same size** - All 256×256px minimum
4. **Same session if possible** - Model consistency
5. **Review together** - Compare all icons side-by-side
6. **Regenerate outliers** - Any that don't match the set
